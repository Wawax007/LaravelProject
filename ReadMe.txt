Bienvenue sur l'application de gestion de stars,

Étapes d'installations :

- Installer wamp64
- Aller dans le dossier C:\wamp64\www (emplacement d'installation)
- Ouvrir une invite de commande dans ce dossier.
- Taper cette commande : git clone https://github.com/Wawax007/LaravelProject.git
- Aller dans le dossier "db" et installer la base de données grâce au script laravelproject_db.sql
- Modifier le fichier .env dans le dossier du projet afin de changer les identifiants de base si nécessaire.
- Pour accéder au projet : http://localhost:8080/LaravelProject/public/index.php
